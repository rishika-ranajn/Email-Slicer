**Email Slicer** is a Python program that takes email addresses as input from the user and separates them into username and domain parts. It allows users to enter multiple emails until they type “exit,” and for each valid email, it displays the extracted username and domain. The program performs basic validation by checking for the presence of “@” and stores all entered data in a list, which is later converted into a pandas DataFrame for a clean tabular display. This program includes concepts like functions, loops, string manipulation, and basic data handling in Python, though it can be improved by adding stronger email validation and saving the data to a file.
